# Specialized Enciphered Access Through Text Steganography

## Project Description
Specialized Enciphered Access Through Text Steganography is a security-based application designed to protect sensitive healthcare information by hiding encrypted data inside text using steganography techniques.

The project focuses on securely sharing patient medical information such as reports, diagnosis details, and health records without exposing the original data to unauthorized users. The confidential information is converted into encrypted text and embedded within a cover text, allowing secure communication between authorized users.

## Features
- Secure medical data transmission
- Text-based steganography for hiding confidential information
- Encryption and decryption of sensitive data
- Prevents unauthorized access and data tampering
- Provides secure patient information sharing

## Technologies Used
- Java
- J2EE
- MySQL
- Apache Tomcat
- Eclipse IDE

## Project Objective
The main objective of this project is to improve healthcare data security by preventing data leakage, unauthorized access, and misuse of sensitive patient information through encryption and steganography methods.
